# 복불복

현재 위치에서 주변 음식점 정보를 확인하고 룰렛 게임을 진행할 수 있는 토이 프로젝트입니다.

## 기술 스택

- TypeScript
- Next.js
- React-Query
- Recoil
- Sass

## 주요 기능

- 주변 음식점 리스트 정보 조회
- 음식점 상세 정보 조회
- 룰렛 게임
